Tools Required:

1. JDK 1.8
2. ApacheTomcat 8.x+


Project Build:

1. Navigate to directory URLProcessor/
2. Run mvn clean package. (This will generate executable jar)


Program Execution:

1. Navigate to directory URLProcessor/target/
2. Run java -jar URLProcessor.jar


Program Output

1. Navigate to directory URLProcessor/target/output/
2. Statistics are generated under file url_statistics.txt
3. Logs are located under URLProcessor/target/output/logs/url_processor.log
